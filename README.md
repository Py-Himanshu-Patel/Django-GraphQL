### Commands and Links

- Setup Virtual Environment.  

    To setup env

    ```bash
    mkdir .venv
    pipenv shell
    ```

    To install the Pipfile

    ```bash
    pipenv install
    ```

- [graphene-django](https://pypi.org/project/graphene-django/) - A Django integration for Graphene.  

    ```bash
    pipenv install graphene-django
    ```
